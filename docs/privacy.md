# TallyBar Privacy Policy

_Last updated: 2026-05-12_

## Summary

TallyBar does not collect, store, transmit, or share any personal data. The app runs entirely on your Mac.

## What the app does

TallyBar reads the contents of your system clipboard (pasteboard) so it can count the characters, words, and lines and display the count in your menu bar.

- Clipboard contents are read in-memory only.
- Clipboard contents are **never** written to disk by TallyBar.
- Clipboard contents are **never** sent over the network.
- TallyBar has no network code, no analytics SDKs, and no third-party services.

## What is stored

TallyBar stores small preferences (e.g., which metric to display, whether to shorten the menu bar label) in the standard macOS user defaults system, on your device. These preferences contain no personal information and never leave your Mac.

## Launch at Login

If you enable "Launch at Login", TallyBar registers itself with macOS using the public `SMAppService` API so the system will start it at next login. No data is transmitted; this is a local OS feature.

## Children's privacy

TallyBar is suitable for all ages and does not knowingly collect data from anyone, including children under 13.

## Changes

If this policy changes, the updated version will be posted at this URL and the "Last updated" date will change.

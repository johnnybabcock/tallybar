# TallyBar Support

## Common questions

**Why does the count not update?**
TallyBar polls the clipboard for changes every ~400 ms. If the count seems stale, copy any text again and the menu bar will refresh.

**Where can I change which metric is shown?**
Click the TallyBar item in the menu bar → "Display in menu bar" → pick Characters / Characters (no spaces) / Words / Lines.

**How do I make the menu bar item narrower?**
Toggle "Shorten label (c / w / l)" in the dropdown.

**Does TallyBar see anything besides the clipboard?**
No. See the [Privacy Policy](privacy.html).
